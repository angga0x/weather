# Weather App

## Description

This is a weather application built with React.

## Technologies Used

*   React
*   Tailwind CSS

## Installation

1.  Clone the repository:

    ```bash
    git clone <repository_url>
    ```
2.  Install dependencies:

    ```bash
    npm install
    ```

## Usage

To run the application:

```bash
npm run dev
```
